# SynapseAI – DeepSeek Inspired Personal AI Chatbot

SynapseAI is a full-stack AI-powered personal chatbot inspired by DeepSeek, developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). The application enables real-time conversational interaction using LLM integration and supports persistent chat history storage for context-aware responses.

## Features
- Multi-session chat support
- Chat history persistence
- LLM API integration
- RESTful API architecture
- Responsive UI using React.js
- Context-based conversational responses

## Tech Stack
- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- API Integration: LLM API
- Authentication
